AI Token Metrix · 隐私政策

隐私政策

生效日期：2026-07-14（第 1.0 版）
最后更新：2026-07-14

AI Token Metrix（"本应用"、"我们"）是一款 macOS 菜单栏工具，通过读取您设备本地存放的 AI 编程工具日志，为您统计 token 用量与预估花费。本《隐私政策》说明本应用处理哪些信息、为什么、以及您享有的权利。

本应用围绕一个核心原则设计：您的使用数据永远不会离开您的设备。

一、个人信息处理者

处理者：W PF，个人独立开发者。
联系方式：tokenusagemonitor@gmail.com

本条明确了：

  • 《中华人民共和国个人信息保护法》（PIPL）意义上的"个人信息处理者"
  • 《GDPR / 英国 GDPR》意义上的"数据控制者"（Data Controller）
  • 《加州 CCPA / CPRA》意义上的"business"

二、本应用处理的信息类别

本应用仅处理以下类别的信息，且仅在您的设备本机处理。本节所列的任何信息均不会传输给我们或任何第三方。

2.1 AI 工具日志文件(仅读取,用户明确授权)

在您通过 macOS 系统的文件选择对话框(NSOpenPanel)显式授权本应用访问某个目录后,本应用会读取您已安装的 AI 编程工具生成的日志文件。根据您启用了哪些 Provider,读取路径包括:

Provider路径
Claude Code~/.claude/projects/*.jsonl
OpenAI Codex CLI~/.codex/sessions/**/rollout-*.jsonl、~/.codex/archived_sessions/
Kimi CLI~/.kimi/sessions/**/wire.jsonl,另读取 ~/.kimi/config.json 获取当前模型名
Kimi Code~/.kimi-code/sessions/**/wire.jsonl
Gemini CLI~/.gemini/tmp/**/chats/*.{json,jsonl}
Google Antigravity~/.gemini/antigravity/.token-monitor/rpc-cache/v1/*.jsonl
OpenCode~/.local/share/opencode/storage/message/**/*.json
GitHub Copilot CLI~/.copilot/otel/*.jsonl(仅当您设置了环境变量 COPILOT_OTEL_ENABLED=true 时)

对于每条日志记录,本应用会提取:

  • Token 计数(输入 / 输出 / 缓存 / 推理)
  • 模型名称(例如 claude-sonnet-4-5)
  • 时间戳
  • 会话 ID —— 由 AI 工具生成,非本应用生成
  • 工作目录(cwd) —— 您运行 AI 工具时所在的目录

关于 cwd:工作目录路径可能包含您的项目名(例如 /Users/您的用户名/Projects/公司机密项目)。根据 PIPL 与 GDPR 的定义,我们将其视为个人信息。此信息仅存储在本地,仅用于在应用内按项目分组展示您的用量,从不传输。

本应用不读取您与 AI 工具的对话内容 —— 只读取上述用量元数据。

2.2 应用配置

存储于您设备本地:

  • 目录授权数据(macOS security-scoped bookmark)
  • 您启用 / 暂停了哪些 Provider
  • 显示币种偏好、周首日偏好、语言偏好
  • 您配置的限额规则(金额上限、告警阈值、静音状态)
  • 菜单栏弹窗的行为偏好

2.3 聚合统计

本应用会从上述原始日志派生出以下数据:

  • 按 (Provider × 模型 × 会话) 分桶的每日汇总
  • 每日总览 —— 一天一条,覆盖所有 Provider

这些聚合数据存储在本应用沙盒容器与 App Group 容器内的本地 SwiftData 数据库中。

2.4 网络请求(仅出站,不发送个人数据)

为了以您偏好的币种显示花费、并使模型定价保持最新,本应用大约每 12 小时向以下公开数据源发起出站 HTTPS 请求:

  • https://models.dev/api.json —— 约 2,900 个 AI 模型的定价数据
  • https://cdn.jsdelivr.net/npm/@fawazahmed0/currency-api/... —— 外汇汇率
  • https://api.frankfurter.dev/... —— 备用外汇汇率(数据源为欧洲中央银行)
  • https://raw.githubusercontent.com/BerriAI/litellm/... 经 jsDelivr / jsdmirror 转发 —— 备用定价数据

这些请求不包含任何个人信息。它们只是下载公开的参考数据。您的用量数据、项目名、会话 ID 或任何关于您的信息从不被上传。

本应用不使用任何分析 SDK、遥测 SDK、崩溃上报 SDK、广告 SDK 或第三方追踪库。

2.5 本地通知

若您配置了限额规则,本应用可能通过 macOS UserNotifications 框架发送本地通知。这些通知完全在您的设备上生成,不通过 Apple Push Notification 服务(APNs)投递。通知内容包含 Provider ID(例如 claude)与限额百分比指示。

本应用仅在您首次保存限额规则时才请求通知权限 —— 而非在启动时。

三、信息用途

  • 计算与展示您的 AI 用量与预估花费于菜单栏界面、详情窗口、Widget
  • 聚合用量为每日汇总,供趋势可视化使用
  • 投递本地通知当您的用量跨过您自己设定的限额阈值时
  • 保留您的偏好跨应用启动持久化

本应用不将您的信息用于任何广告、画像、销售、跨场景行为广告或自动化决策目的。

四、数据存储位置

所有信息仅存储在您的设备上,位于您 macOS 用户账户下的以下位置:

  • 沙盒容器:~/Library/Containers/com.wpf.aitokenmetrix/Data/
  • App Group 容器(与 Widget 扩展共享):~/Library/Group Containers/group.com.wpf.aitokenmetrix/

这些目录受 macOS 沙盒保护,其他应用无法访问。

系统级 iCloud 备份:若您设备的系统设置启用了对本应用数据的 iCloud 备份,则部分数据可能被包含在 Apple 对您设备的 iCloud 备份中。此情况下 Apple 作为受托处理者,适用 Apple 自身的隐私政策 —— 与本政策无关。

未来功能 —— iCloud 同步:未来版本可能通过 iCloud(CloudKit 私有数据库)在您自己的 Apple 设备之间同步"每日总览"表。若启用此功能,同步的数据将保留在您自己的 iCloud 账户中;Apple 将代表您处理这些数据,我们无法访问。此功能目前尚未启用;启用时本《隐私政策》将同步更新,并在应用内提前通知您。

五、数据共享与第三方

我们不向任何第三方出售、出租、共享或转让您的个人信息。

唯一的第三方交互是第 2.4 节列出的出站 HTTPS 请求,用于下载公开参考数据(定价表、汇率)。这些请求不携带任何关于您或您使用行为的信息。

本应用未集成任何分析、广告、社交登录或崩溃上报服务。

六、数据保存期限

  • 本地数据在您的设备上无限期保留,直到您删除。
  • 您可随时通过在应用设置中禁用相应 Provider,或撤销目录授权,停止数据收集。

6.1 如何删除您的数据

由于所有数据均为本地存储,您可以通过以下任一方式删除:

  1. 卸载应用:将本应用拖到废纸篓。此操作删除应用二进制,但可能在 ~/Library/Containers/ 与 ~/Library/Group Containers/ 中留下数据。
  2. 手动删除数据目录:移除第四节列出的目录。

当前局限:当前版本尚未在应用内提供"删除全部数据"按钮。这是我们计划改进的项目(见第七节 —— 删除权)。在该改进上线前,请使用上述任一方式。

您根据 PIPL(中国)、GDPR(欧盟 / 英国)、CCPA / CPRA(加州)及类似法律享有以下权利:

权利行使方式

八、儿童隐私

本应用面向使用 AI 编程工具的成年开发者。我们不会有意处理 13 周岁以下(或您所在司法辖区规定的等效最低年龄)儿童的信息。若您认为有儿童使用了本应用,请联系我们;由于所有数据均在儿童设备本地,删除操作由儿童 / 监护人直接控制。

九、安全措施

  • 所有数据存储于 macOS 沙盒与 App Group 容器中,受 macOS 用户级访问控制保护
  • 所有出站网络请求均使用 HTTPS
  • 本应用当前不收集任何凭证或身份令牌(未来版本若支持官方云端 API,凭证将存储在 macOS Keychain 中 —— 绝不明文存储)
  • 未嵌入任何第三方分析或追踪 SDK

十、跨境数据传输

由于所有个人数据均在您的设备上本地处理,不传输给我们或任何第三方,本《隐私政策》下不存在跨境数据传输。

第 2.4 节所述出站请求下载来自可能托管于不同区域 CDN 的公开参考数据;由于这些请求不携带个人信息,它们不构成 GDPR / PIPL 意义上的个人信息跨境传输。

未来 iCloud 同步功能:若您在未来版本中启用此功能,同步操作通过您自己的 iCloud 账户完成。Apple 存储您 iCloud 数据的区域由您 Apple ID 的账户设置和 Apple 自身的隐私政策决定,与我们无关。

  • 更改本文档顶部的"生效日期"与"最后更新"字段
  • 同步更新嵌入应用内的副本与托管于 https://pf-w.github.io/ai-token-metrix-docs/privacy-policy/ 的副本
  • 对于重大变更(例如启用 iCloud 同步、新增数据类别),我们将在变更生效前在应用内呈现通知
